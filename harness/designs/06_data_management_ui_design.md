# 06_data_management_ui_design.md

## 页面概述
**页面名称**: DataManagement
**文件位置**: `frontend/src/pages/DataManagement.tsx`
**路由**: `/data-management`

## 设计风格
- 保持与项目一致的 Tailwind CSS 样式
- 使用现有的设计组件：Card, Button, Select, Input
- 主色调：primary-500 / primary-600
- 强调色：emerald（成功）、red（错误/删除）

## 组件设计

### 1. 上传区域
```
┌─────────────────────────────────────────────────────────────┐
│  border-2 border-dashed border-slate-200 rounded-xl p-8    │
│                                                             │
│         📤 Upload icon (w-12 h-12)                         │
│                                                             │
│         拖拽文件或点击上传                                    │
│         支持 CSV 格式                                        │
│                                                             │
│  input[type=file] (hidden)                                 │
└─────────────────────────────────────────────────────────────┘
```

**状态**:
- Default: 灰色虚线边框
- Hover: primary-300 边框
- Uploading: 显示进度条
- Success: 显示成功提示 + 文件信息

### 2. 数据集表格

| 列 | 样式 | 说明 |
|----|------|------|
| 文件名 | font-medium, truncate, max-w-[200px] | 显示文件名 |
| 大小 | text-sm text-slate-500 | 格式化显示 (KB/MB) |
| 行数 | text-right | 数字显示 |
| 列数 | text-right | 数字显示 |
| 上传时间 | text-sm text-slate-500 | 相对时间或日期 |
| 操作 | flex gap-2 | 预览/统计/删除按钮 |

**操作按钮样式**:
- 预览: variant="secondary", size="sm"
- 统计: variant="secondary", size="sm"
- 删除: variant="danger", size="sm"

## 响应式设计
- Desktop (lg+): 表格完整展示
- Tablet/Mobile: 表格横向滚动，操作按钮紧凑排列

## 状态管理

```typescript
interface DataManagementState {
  // 数据
  files: DataFile[]
  selectedFile: DataFile | null
  
  // 上传
  uploading: boolean
  uploadProgress: number
  
  // 详情面板
  detailTab: 'preview' | 'stats'
  previewData: PreviewResponse | null
  statsData: StatsResponse | null
  loadingDetail: boolean
  
  // 删除确认
  deleteConfirm: number | null  // file id
  
  // 错误
  error: string | null
}
```

## 错误处理
- 上传失败: 显示红色错误提示，保留用户输入
- 加载失败: 显示重试按钮
- 网络错误: 显示友好提示

## 空状态
- 无数据集时:

