# ML All In One 前端 - 需求文档

**项目名称：** ML All In One 现代UI前端  
**创建日期：** 2026-04-09  
**文档版本：** v1.0

---

## 1. 项目概述

为 ML All In One 机器学习平台开发一套简约现代风格的 Web 前端界面，替代现有的 Gradio 界面，提供更好的用户体验。

**目标用户：** 数据科学家、机器学习工程师、业务分析人员

**核心价值：**
- 简化机器学习模型训练和部署流程
- 提供直观的数据可视化
- 支持实验追踪和模型对比

---

## 2. 功能需求

### 2.1 仪表盘（Dashboard）
- 显示系统统计信息（总训练次数、活跃实验数、运行时长、最优模型指标）
- 快速入口（上传数据集、自动机器学习、模型预测）
- 最近模型列表（名称、准确率、数据集、时间）

### 2.2 模型训练（Training）
- **数据上传**：支持 CSV 文件拖拽上传，显示文件信息
- **任务配置**：
  - 任务类型选择（分类/回归）
  - 目标列指定
  - 模型选择（RandomForest、XGBoost、LightGBM、LogisticRegression）
- **训练控制**：开始/停止训练按钮
- **实时进度**：训练进度条、当前迭代、准确率、已用时间

### 2.3 实验记录（Experiments）
- 实验列表表格（名称、状态、准确率、F1分数、耗时、时间）
- 状态标签（完成/失败/运行中）
- 实验详情面板（模型参数、数据集信息）
- 刷新功能

### 2.4 数据预处理（Preprocessing）【后续阶段】
- 数据预览和基本统计
- 缺失值处理
- 特征归一化
- 数据集划分

### 2.5 模型推理（Inference）【后续阶段】
- 模型选择
- 批量预测
- 结果导出

---

## 3. 非功能需求

### 3.1 性能要求
- 首屏加载时间 < 2秒
- API 响应时间 < 500ms
- 页面切换流畅，无明显卡顿

### 3.2 兼容性
- 现代浏览器（Chrome/Firefox/Safari/Edge 最新两个版本）
- 响应式设计，支持 1024px 以上屏幕

### 3.3 可用性
- 清晰的操作引导
- 友好的错误提示
- 加载状态指示

---

## 4. 技术约束

### 4.1 技术栈
- **前端框架**：React 18 + TypeScript
- **构建工具**：Vite
- **样式方案**：TailwindCSS
- **图表库**：Recharts
- **状态管理**：TanStack Query
- **HTTP客户端**：Axios
- **图标库**：Lucide React

### 4.2 后端接口
- FastAPI 后端（端口 8000）
- RESTful API 风格
- JSON 数据交换

### 4.3 现有系统
- mlkit 核心库位于 `src/mlkit/`
- 实验数据存储于 `experiments/` 目录
- 模型 checkpoint 存储于 `checkpoints/` 目录
